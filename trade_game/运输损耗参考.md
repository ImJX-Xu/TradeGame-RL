# 运输损耗参考

## 一、损耗计算公式

实际单件损坏概率：

```
p = (base_rate + taiwan_bonus) × multiplier × random_factor
```

其中：
- **base_rate**：商品 `transport_loss_rate`（基础损耗率）
- **taiwan_bonus**：途经台北或高雄时 **+1%（0.01）**
- **multiplier**：按商品类别不同（见下文）
- **random_factor**：每趟运输随机 **0.9～1.1**

---

## 二、损耗乘数（multiplier）

| 类别 | 标签 | 乘数公式 | 说明 |
|------|------|----------|------|
| **易损** |  electronics | `1 + km / 1000` | 与里程成正比 |
| **轻工** | light_industry | `1 + (km / 1000) / 3` | 里程的 1/3 强度 |
| **民用** | base | `1 + (km / 1000) / 3` | 同轻工 |
| **生鲜** | perishable | `(1 + days / 5) × (1 + S × (days/10)²)` | 与运输天数相关，S 为老化强度 |

**常量：**
- `LOSS_REF_KM = 1000`：里程参考值
- `LOSS_REF_DAYS = 5`：生鲜时间参考值

## 五、其他因素

| 因素 | 说明 |
|------|------|
| **台湾路线** | 起/终点含台北或高雄时，`base_rate + 0.01` |
| **随机因子** | 每趟运输整体乘 0.9～1.1 |
| **二项分布** | 按单件损坏概率 p，对每批货物采样实际损耗数量 |
